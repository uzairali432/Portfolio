# 🚀 Personal Portfolio Website

A modern, responsive, and animated personal portfolio website built using **Next.js**, **Tailwind CSS**, and **Framer Motion**.  
This portfolio showcases my skills, projects, and experience with smooth animations and a clean UI.

---

## ✨ Features

- ⚡ Built with **Next.js** for fast performance and SEO
- 🎨 Styled using **Tailwind CSS**
- 🎬 Smooth animations with **Framer Motion**
- 📱 Fully responsive (Mobile, Tablet & Desktop)
- 🧩 Clean and reusable components
- 🌙 Modern UI with attractive color combinations
- 🚀 Optimized for performance and accessibility

---

## 🛠 Tech Stack

- **Framework:** Next.js  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **Language:** JavaScript (JSX)
